## 1. 분석 절차 수립



**분석 모형 선정 필요성**

1. 의사 결정
2. 불확실성 해소
3. 요약
4. 인과관계 파악
5. 예측



**분석 모형 선정 프로세스**

1. 문제요건 정의 또는 비즈니스 이해에 따른 대상 데이터 선정과 분석 목표/조건 정의 
2. 데이터 수집, 정리 및 도식화
3. 데이터 전처리
4. 최적의 분석 모형 선정



**분석 모형 종류***

1. 예측 분석 모형
   - 어떤 일들이 발생할 것인가?
2. 현황 진단 모형
   - 과거에 어떠한 상황이 왜 어떻게 일어났는가?
3. 최적화 분석 모형
   - 어떻게 하면 원하는 결과가 일어날 수 있을까?
