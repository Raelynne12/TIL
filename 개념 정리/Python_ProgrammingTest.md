### 5) 반복제어문에 대한 잘못된 설명

\>> continue : 특정 조건일 경우 반복문 건너뛰기

\>> break : 특정 조건일 경우 반복문 종료

\>> exit : 특정 조건일 경우 프로그램 종료

\>> pass : 문법적으로 오류 발생시키지 않게 하기 위해 자리 채우기

\>> plt.scatter(x, y, kind = 'kde')		(scatter plot은 산점도, kind = kde는 커널 밀도 함수)
